# Как запустить тесты и посмотреть отчет Allure (Урок 10)
В этом проекте я подключила генерацию отчетов Allure.
## Шаг 1. Запуск тестов и сбор данных
Чтобы запустить тесты и собрать всю информацию об их 
прхождении в специальную папку allure-results, откройте терминал в
VS Code и выполните команду:
```pytest lesson_10/ --alluredir=allure-results```
## Шаг 2. Открытие отчета в браузере
Чтобы превратить эти файлы в графический отчет с графиками и
шагами, введите в терминал команду:
```allure serve allure-results```
